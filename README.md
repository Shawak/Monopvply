## Installing using docker

### Requirements

  * docker
  * docker-compose

### Installing

    git clone http://gitlab.syc.wtf/Syc/Monopvply.git monopvply -b <branch>
    cd monopvply && docker-compose up --build
    
### Updating

    git pull
